# kuma-code portfolio site

複数ページ構成のポートフォリオサイト（Bootstrap 5）。

## ページ
- index.html（トップ）
- works.html（実績）
- pricing.html（料金）
- about.html（自己紹介）
- contact.html（問い合わせ）

## 画像
- 写真：Unsplash（フリー素材URL）
- 実績サムネイル：ページスクリーンショット生成サービス（thum.io）を使用
  - https://kuma-code215.github.io/portfolio-farm/
  - https://kuma-code215.github.io/portfolio-beauty/

公開時に外部画像をローカル運用にしたい場合：
1) サムネイルを画像として保存（png/jpg）
2) works/index の img src をローカルパスに変更
